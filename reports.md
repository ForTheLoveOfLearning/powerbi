## Visualizations
| Main Visual Category | Visualization Name   | Description                                                                                                    | Other Notes                                                                                                                                   |
|----------------------|-----------------------|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| Chart                | Stacked Bar           | Display relationships between values.                                                                         |                                                                                                                                               |
| Chart                | Stacked Column        | Display relationships between values within categories, or changes over time.                                   |                                                                                                                                               |
| Chart                | Clustered Bar         | Display multiple relationships between values.                                                                 |                                                                                                                                               |
| Chart                | Clustered Column      | Display multiple relationships between values within categories, or changes over time.                           |                                                                                                                                               |
| Chart                | 100% Stacked Bar     | Display relationships between values within categories (or changes over time) as related to the whole.         |                                                                                                                                               |
| Chart                | 100% Stacked Column  | Display relationships between values as related to the whole.                                                  |                                                                                                                                               |
| Chart                | Line                  | Show the progression of values over time.                                                                     |                                                                                                                                               |
| Chart                | Area                  | Show relationships between values over time.                                                                   |                                                                                                                                               |
| Chart                | Stacked Area          | Show relationships between values over time as part of a whole.                                                |                                                                                                                                               |
| Chart                | Line and Stacked Column | Display relationships between values within categories, or changes over time, with a constant line.           |                                                                                                                                               |
| Chart                | Line and Clustered Column | Display relationships between values within categories, or changes over time, with a constant line.         |                                                                                                                                               |
| Chart                | Ribbon                | Similar to a Line chart, but with a 3-D display and additional data options.                                    |                                                                                                                                               |
| Chart                | Funnel Chart          | Display a process that has stages and items flow sequentially from one stage to the next.                      |                                                                                                                                               |
| Chart                | Waterfall             | Displays the cumulative effect of positive or negative values.                                                 |                                                                                                                                               |
| Chart                | Scatter               | Displays values for two variables within a dataset.                                                            |                                                                                                                                               |
| Chart                | Pie                   | Displays values as part of the whole.                                                                         |                                                                                                                                               |
| Chart                | Donut                 | Also displays values as part of the whole but allows comparison of more than one data series.                  |                                                                                                                                               |
| Maps                 | Treemap               | Display large amounts of hierarchical data, showing proportions between each part of the whole.                | Present a large number of values that can't be effectively shown with a bar chart. Illustrate the proportions between each part and the whole. |
| Maps                 | Map                   | The standard map visual.                                                                                      | Requires internet connection                                                                                                                  |
| Maps                 | Filled Map            | The rectangles are arranged by size with the largest branch node at the top left and the smallest branch at the bottom right. The arrangement of the leaf nodes within each branch follows the same order |                                                                                                                                               |
| Maps                 | ArcGIS Map            | Map visual with enhanced: mapping capabilities, demographic data, location analytics, and spatial analysis tools. | Users must accept terms and privacy on the consent dialog before using it                                                                    |
| Maps                 | Azure Maps            | The Azure Maps Power BI visual provides a rich set of data visualizations for spatial data on top of a map. It connects to Azure to retrieve images to help render images and coordinates. | Requires extra permissions to be enabled at the org level                                                                                   |
| Metric               | Gauge                 | Display a single value in relation to a target or gauge. Used to show the progress towards a measurable goal.  |                                                                                                                                               |
| Metric               | Single Number         | Displays a single fact or data point.                                                                         |                                                                                                                                               |
| Metric               | Single Number (New)  | Displays a single fact or data point.                                                                         |                                                                                                                                               |
| Metric               | Multi Row             | Displays one or more data points/facts.                                                                       |                                                                                                                                               |
| Metric               | KPI Card              | Displays the amount of progress made towards a measureable goal. Measures the distance to a metric.           |                                                                                                                                               |
| Filter               | Slicer                | On page visual version of a filter using a column of a data table to filter the data by with options to display the slicer and options to select one, multiple and all data to filter by. |                                                                                                                                               |
| Filter               | Slicer (New)         | Like a slicer but with added customizations to enhance the appearance and interaction.                         |                                                                                                                                               |
| Table                | Table Visual          | Displays data across multiple dimensions, automatically aggregating data and allowing drill down capabilities. |                                                                                                                                               |
| Table                | Matrix                | Supports two dimensions with flat data, meaning duplicate values are displayed and not aggregated.             |                                                                                                                                               |
| Languages to Create  Visual | R                   |                                                                                                                |                                                                                                                                               |
| Languages to Create  Visual | Python              |                                                                                                                |                                                                                                                                               |

| Specialised AI Visuals | Q&A                  | AI visual that uses Natural language questions to parse the data and respond with visualizations.               |                                                                                                                                               |
| Specialised AI Visuals | Smart Narrative      | A dynamic visual that produces information about the data in the report.                                        |                                                                                                                                               |
